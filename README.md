# Password Management Tool
# Author: Abhinav Silwal

A simple Python GUI application built with Tkinter that allows users to:
- Check the strength of a password
- Detect common and weak passwords
- Generate strong random passwords
- Copy generated passwords to the clipboard

## Features
- Password strength checking using entropy
- Common password detection
- Secure password generator
- Visual strength bar
- Simple and clean GUI

## Technologies Used
- Python
- Tkinter

## How to Run
1. Make sure Python is installed on your device
2. Clone or download this repository
3. Open a terminal in the project directory
4. Run the program using: python password_management_tool.py
